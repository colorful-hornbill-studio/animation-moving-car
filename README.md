## Getting Started

### Requirements

1. NetBeans IDE 8
2. JDK 8
3. JRE 8

### Installation

1. Extract file ```animation-moving-car-master.zip``` in your NetBeans projects folder.
2. Rename folder ```animation-moving-car-master``` to ```animation-moving-car```.

### Usage

1. Open project ```animation-moving-car``` in NetBeans IDE.
