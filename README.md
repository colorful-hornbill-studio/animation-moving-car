## Getting Started

### Prerequisites

1 NetBeans IDE 8
2 JDK 8
3 JRE 8

#### Installation

1 Extract file ```animation-moving-car-master.zip``` in your NetBeans projects folder

### Usage

1 Open extracted folder
2 Go to ```dist``` folder
3 Select file ```Cartoon.jar```
